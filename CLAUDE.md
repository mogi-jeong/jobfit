# 잡핏 관리자 웹 — 개발 지침

## 프로젝트 개요
잡핏(JobFit) 단기 알바 알선 플랫폼의 관리자 웹 프로토타입.
기획안 v1.3 기반. 김진호 사장 1인 운영 → 친척동생과 공동 제작.

## 기술 스택
- HTML/CSS/JavaScript (순수 구현, 프레임워크 없음)
- Kakao Maps JavaScript API (services, drawing 라이브러리)
- 배포: GitHub Pages (mogi-jeong.github.io)
- 실제 앱 개발 시: Flutter 3.x + Supabase

## 코드 스타일
- 딥 네이비 #1B3A6B (사이드바, 헤더)
- 인디고 #1E40AF (강조)
- 로열 블루 #2563EB (CTA 버튼)
- 성공 #22C55E / 경고 #F59E0B / 에러 #EF4444
- 버튼 최소 36px (모바일은 48px)
- border-radius: 8px (기본), 12px (카드)
- 한글 UI, 한국어 주석

## 용어 체계 (매우 중요)
- 내부 명칭 (DB/관리자 화면): "파트너사" / "근무지"
- 알바생 UI: "업종" / "지역"
- "블랙리스트" 쓰지 말 것 → "협의대상"으로 통일

## 권한 체계 (3단계)
- 마스터: 전체 권한 + 근무지 등록/GPS 영역 설정 (사장님 1명)
- 관리자 1등급: 전 근무지
- 관리자 2등급: 담당 근무지만

## 현재 작업 상태
- 파일: index.html (근무지 관리 페이지)
- 버전: v0.3.1
- 완성 기능: 홈 대시보드, 근무지 관리, GPS 다각형 영역 설정, 주소 검색
- 미구현: 공고 관리, 신청 승인, 근무자 관리, 관제 시스템, 협의대상, 포인트, 문의, 관리자 계정
