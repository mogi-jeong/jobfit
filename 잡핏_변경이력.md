- 적용: 검색 6개 (문의/감사로그/관리자/협의대상/근무자/이력) + 알림 발송 모달 (제목/본문)
- 공고 등록 폼 텍스트 필드 3개는 `__jfSetText`로 분리 (state만 update, re-render 안 함)

**홈 v2 (재설계 후보)** — 사이드바에 [홈 (재설계)] 별도 메뉴로 비교 가능
- 3단 위계: 처리 필요 → 오늘 한눈에 → 디테일
- 큰 도넛 차트(출근율) + 핵심 KPI 3개 + 보조 칩 5개 + 데모 가이드 접기

**공고 복제 기능 제거** — 즐겨쓰는 템플릿이 더 유연하게 대체
